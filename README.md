# new-video-project

## これは何

Premiere Pro のプロジェクトで利用するディレクトリ管理テンプレートです。

## ディレクトリ構成

```
new-video-project/
  |- assets/ #イメージやオーディオ素材などを入れておく
  |  |- audio/ #オーディオ素材を入れておく
  |  |- images/ #イメージ素材を入れておく
  |- docs/ 絵コンテなど資料を入れておく
  |- footage/ #ビデオ素材を入れておく
  |- renderd/ #出力したビデオを入れておく
```

## 使い方

```
$ git clone git@github.com:kazoo0217/new-video-project.git
$ mv new-video-project rename-project-name
```
